## Run Locally   return

Clone the project
(`git clone git@github.com:Harrytimbog/teddies_shop.git`)

Go to the project directory
(`cd teddies_shop`)

Install dependencies
(`npm install`)

Install gems
(`bundle install`)

Start Server
(`rails s`)
